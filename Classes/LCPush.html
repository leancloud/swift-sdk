    application

    The application, default is LCApplication.default.
    data

    The body data of the push message.
    query

    The query condition, if this parameter be set, then channels will be ignored.
    channels

    The channels condition, if query be set, then this parameter will be ignored.
    pushDate

    The date when to send.
    expirationDate

    The expiration date of this push notification.
    expirationInterval

    The expiration interval from pushDate of this push notification.
    extraParameters

    The extra parameters, for some specific configuration.

    Boolean result, see LCBooleanResult.
